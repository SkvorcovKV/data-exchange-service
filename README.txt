================================================
Сервис обмена данными (Data Exchange Service) v1.0
================================================

НАЗНАЧЕНИЕ:
Сервис предназначен для обмена данными с СКУД "ЭНТ Контроль доступа"
через программу "Обмен данными" версии 1.3.

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Операционная система: Windows 10/11 (64-bit), Windows Server 2019/2022
- Процессор: 64-разрядный, 1 ядро, 1 ГГц (мин.), 2 ядра, 2 ГГц (реком.)
- Оперативная память: 512 МБ (мин.), 1 ГБ (реком.)
- Свободное место: 200 МБ (мин.), 500+ МБ (реком.)
- Microsoft Visual C++ Redistributable 2015-2022 (x64)

УСТАНОВКА:
1. Распакуйте архив в любую папку (например, C:\ENT Exchange Service)
2. Установите Microsoft Visual C++ Redistributable по ссылке:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

ЗАПУСК:
1. Запустите файл ent-exchange-service.exe
2. В консоли появится сообщение: 
   "Сервис запущен. Веб-интерфейс доступен по адресу: http://localhost:7556"
3. Откройте браузер и перейдите по указанному адресу

НАСТРОЙКА:
- Файл конфигурации config.ini создается автоматически при первом запуске
- Основные настройки можно изменить через веб-интерфейс (вкладка "Настройки")

ОСТАНОВКА:
- Закройте окно консоли или нажмите Ctrl+C

ПОРТЫ:
- По умолчанию используется порт 7556
- При изменении порта в настройках требуется перезапуск сервиса

ЛОГИ:
- Файл лога: service.log в папке с программой
- Расширенное логирование включается в настройках
